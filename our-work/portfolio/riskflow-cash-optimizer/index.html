Riskflow Cash Optimizer

Northgate, South Africa

A financial management service for beginners and novices alike, to log on using any device via the web to manage financial needs anywhere and anytime.

It has an easy to use interface that allows a user to add their values and assumptions for income, expenses, assets, debt or resources. The results are calculated and displayed on income statements, balance sheets and cash flow statements.

Client Requirements

A financial management service for beginners and novices alike, to log on using any device via the web to manage financial needs anywhere and anytime.

Features

Easy to Use Navigation

Navigation is simple and easy. Labels and descriptions have been used to keep everything and consistent.

Extended Subscription

A user can extend their subscription and view how many days are left before it expires on the Profile page.

Multiple Values / Multiple Entries

The App gives the user the ability to add multiple values to get more accurate estimates. For example, Multiple Properties or Vehicles.

Various Currencies

A preferred currency may be selected from the Profile page via the drop-down list.
